# Customer Behavior Analysis

## Objective
Analyze customer transactions to identify customer segments, purchasing behavior, and churn risks.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- RFM Analysis
- K-Means Clustering
- Customer Segmentation
- Churn Analysis
- Business Recommendations

## Results
- Identified customer segments using RFM and K-Means.
- Visualized purchasing patterns and churn.
- Provided five actionable recommendations to improve customer engagement.
